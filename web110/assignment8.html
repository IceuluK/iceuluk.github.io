Activity: Learning about CSS
In this session, we learned about Cascading Style Sheets. It is a style sheet language used to describe how elements are presented in a document written in a markup language like HTML.
I learned about concepts of external (can change style of an entire website), internal (change style of one page), and inline CSS (change style of one element). I practiced applying those styles to the web pages containing my assignments.
I also learned how to change background color, text color, margins, font type and size.
All this knowledge will helpful:
1) in my personal life - if/when I get to creating a blog, I could control the appearance of it myself;
2) in my professional life - I need to be able to read CSS to examine web pages.
I am looking forward to practice CSS and HTML more.
