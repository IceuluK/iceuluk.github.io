Activity: Compare browser HTML5 compatibility.
HTML stands for HyperText Markup Language and is the most popular markup language for creating web pages. It has different versions, with HTML5 being the latest version of HTML which supports more features (multimedia, new tags and elements, new Application Programming Interfaces (APIs)).
I reviewed the offered resource (www.html5accessibility.com) and learned that different bworsers have different HTML5 compatibility.The tool showed that Chrome 84 on Windows 10 has 98.5% HTML5 accessibility.
Using the second tool, "HTML5 Test: How well does your browser support HTML5?", I found "your browser" tab to see how well my browser, Google Chrome, supports HTML5, and it scored 528 out of 555 points. For comparison, Opera is at close second with 518 points, and Edge browser showed to score 429 out of 555 points.
With this activity, I learned how to check a browser's HTML5 compatibility. This knowledge will help me choose the best browser suited for my needs, both personal and at work.

Activity: Cite a web page.
I reviewed the Cite this page link and tried to cite a Wikiversity page from another class I am taking, Windows Server Administration/Installation. I also reviewed the Cite This Page for Internet Fundamentals/Search Engines.
In both cases, pages contained bibliographic details like Page Name, Author, Publisher (Wikiversity), Date of last revision, Date retrieved, URL, Primary contributors, Page Version ID.
Different citation styles had different details included, for example:
APA style:Internet Fundamentals/Search Engines. (2021, November 18). Wikiversity. Retrieved 16:38, November 18, 2021 from [link to the page]
Chicago style: Wikiversity contributors, "Internet Fundamentals/Search Engines," Wikiversity, [link to the page] (accessed November 18, 2021).
With this activity, I learned to cite a web page, and explored different citation styles. This knowledge will help me in preparing reports at work or written assignments for school classes.
