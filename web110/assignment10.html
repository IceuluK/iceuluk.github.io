Activity: Introduction to Javascript
Summary:
In this session, I was introduced to Javascript.
Javascript is the programming language of the Web. It doesn't need to be compiled - modern web browsers support JavaScript with built-in interpreters.
After completing activities of the session, I learned how to insert scripts to display text, current date, or various canvas.
This knowledge will help me:
- in my personal life: when I need interactive elements in my future online blog.
- in my professional life: although I am not going to be a front-end web developer, I will sometimes need to inspect web pages and observe scripts running, to see if there's anything suspicious or hidden.
For those purposes, I hope to continue my exploration of Javascript and become more proficient in it.
