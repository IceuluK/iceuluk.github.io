Assignment 12

Activity: Basic SQL statements


Select all customers

  • SELECT * FROM Customers;

Select customers from USA

  • SELECT * FROM Customers
    WHERE Country='USA';

Count number of products (count number or rows that match ProductID)

  • SELECT COUNT(ProductID)
    FROM Products;

Insert a new record in the Customers table

  • INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country)
    VALUES ('AK Sweets','Aisuluu K','250 W Dundee Rd','Wheeling','60090','USA');

Update the first customer's contact name and city in Customers table

  • UPDATE Customers
    SET ContactName='Hugo Stiglitz', City='Berlin'
    WHERE CustomerID=1;
